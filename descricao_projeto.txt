Olá! Este é o meu projeto para a solução do teste técnico e eu desenvolvi o sistema em Python, (linguagem qual estou mais familiariazado) V - 3.14. 
Eu organizei tudo em módulos para facilitar manutenção e a escalabilidade, usei também a 
biblioteca Rich para criar uma interface no terminal mais visual, com tabelas e painéis 
estilizados, então nada de interface gráfica, tudo via CLI. 

A arquitetura é simples: 
- Um arquivo para dados [data_store.py]
- Outro para serviços [services.py]
- O principal para interação com o usuário [main.py]

A lógica que eu usei gera alunos em massa e distribui registros aleatórios entre turmas, simulando um ambiente real.
Também implementei validações e tratamento de erros para evitar crashes e melhorar a experiência.
